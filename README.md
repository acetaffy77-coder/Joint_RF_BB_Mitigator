# Joint_RF_BB_Mitigator
本项目基于 IEEE Transactions on Microwave Theory and Techniques (TMTT) 的论文：    "Joint Mitigation of Nonlinear RF and Baseband Distortions in Wideband Direct-Conversion Receivers" (作者：Michael Grimm, Mikko Valkama 等, 2014)，  通过 MATLAB 搭建了仿真场景，构建了论文中提出的自适应数字缓解结构， 复现了论文中的Fig.7(a)中双音信号的自适应滤波结果并分析了平均非线性失真抑制比。

仿真平台：MATLAB R2024b


---

## 1. 系统参数与输入信号模型

系统采用双音测试信号作为激励输入，以激发并观察系统的三阶互调失真（IMD3）等非线性产物。

| 参数名称 | 符号 | 数值 | 单位 |
| :--- | :---: | :--- | :--- |
| 采样率 | fs | 25e6 | Hz |
| 信号时长 | T | 0.5 | ms |
| 双音频率1 | f1 | 2.4e6 | Hz |
| 双音频率2 | f2 | 2.8e6 | Hz |
| 中心频率 | fc | 2.6e6 | Hz |
| 总输入功率 | P_input_dBm | -25 | dBm |

输入的基带双音信号数学表达式为：
$$x_{bb}(t) = A (e^{j 2\pi f_1 t} + e^{j 2\pi f_2 t})$$

## 2. 级联非线性失真建模

信号在接收/发射链路中经历了三个主要阶段的非线性失真，模型中最后引入了加性高斯白噪声（AWGN，信噪比为 **61 dB**）。

### 2.1 射频（RF）非线性
利用多项式模型对RF前端的非线性进行建模：

$$y_{rf} = a_1 x_{bb} + a_2 x_{bb} |x_{bb}|^2$$


### 2.2 I/Q 增益与相位失配
模拟正交混频器引入的失配，增益不平衡度 $g_m = 0.99$，相位偏移 $\phi_m = 0.0628$ rad：

$$y_{tilde\_I} = \text{Re}\{y_{rf}\}$$
$$y_{tilde\_Q} = g_m \sin(\phi_m)\text{Re}\{y_{rf}\} + g_m \cos(\phi_m)\text{Im}\{y_{rf}\}$$

### 2.3 基带（BB）非线性
基带放大器等器件引入的进一步非线性，分为I路和Q路独立处理：

$$y_{BB\_I} = a_3 y_{tilde\_I} + a_4 y_{tilde\_I}^3$$
$$y_{BB\_Q} = a_3 y_{tilde\_Q} + a_4 y_{tilde\_Q}^3$$

最终基带信号为 $y_{BB} = y_{BB\_I} + j y_{BB\_Q}$。

## 3. 数字滤波器设计

为了在数字域提取主信号与失真分量以供后续自适应对消，系统设计了两个高阶FIR滤波器（滤波器阶数均为 **500阶**）：

**复带通滤波器 ($H_{C\_{bpFilt}}$)**：
* 由通带截止频率为 **0.3 MHz** 的实低通滤波器经过中心频率 $f_c$ 的数字频移获得。
* **作用**：从受损信号 $y_{BB}$ 中提取期望的线性主信号分量 $x_{hat}$。
    
**实带阻滤波器 ($H_{R\_{bsFilt}}$)**：
* 阻带范围为 **2.3 MHz** 至 **2.9 MHz**。
* **作用**：用于滤除基函数构造过程中落入主信号频带内的分量，确保仅对带外非线性失真进行估计和对消。

*(注：系统通过补零延迟 `gd_bp` 和 `gd_bs` 实现了各支路信号的严格时间对齐)*

## 4. 自适应非线性缓解算法

算法的核心是通过构造非线性基函数，并利用自适应滤波器寻找最佳权重，以逼近并抵消接收信号中的失真分量。

### 4.1 基函数构造

### 4.2 权重更新 (NLMS)
采用归一化 LMS 算法进行权重更新。误差信号及权重迭代公式如下：

$$e(n) = \mathbf{w}^H \mathbf{s}(n)$$
$$x_{est}(n) = d(n) - e(n)$$
$$\mathbf{w}(n+1) = \mathbf{w}(n) + \text{diag}(\mathbf{\mu}_N) x_{est}^*(n) \mathbf{s}(n)$$

其中，
$\mathbf{\mu}_N$ 为考虑了防止除零项（
$af$ 参数）的自适应步长向量。最终的缓解信号为延迟对齐后的主信号与补偿信号之和。

## 5. 性能评估

系统主要通过对比非线性缓解前后的**功率谱密度 (PSD)** 来直观评估算法性能。

* **频谱改善**：在绘图阶段，利用 `periodogram` 函数计算信号末端 **2048** 个采样点的频谱。缓解后（蓝色曲线）的带外三阶及高阶互调产物以及镜像分量相较于缓解前（红色曲线）显著降低。
* **ANDR（平均非线性失真抑制比）**：系统定义了三段主要失真信号所在频段，计算缓解前后带内残留功率的均值之比：
  
$$\text{ANDR} = 10 \log_{10} \left( \frac{\text{缓解前带内平均功率}}{\text{缓解后带内平均功率}} \right)=32.43dB$$
